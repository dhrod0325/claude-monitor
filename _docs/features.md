# 기능 가이드

## 화면 구성

Claude Monitor는 좌측 사이드바를 통해 주요 기능에 접근합니다.

| 메뉴 | 설명 |
|------|------|
| Sessions | 세션 모니터링 |
| Usage | 사용량 대시보드 |
| Analysis | 프롬프트 분석 |
| Work Analysis | 업무 분석 |

## 1. 세션 모니터링

Claude Code에서 생성된 세션을 실시간으로 확인합니다.

### 프로젝트 선택

1. 좌측 상단 드롭다운에서 프로젝트 선택
2. 해당 프로젝트의 세션 목록 표시
3. 프로젝트 경로는 Base64로 인코딩된 ID로 관리

### 세션 목록

각 세션 카드에 표시되는 정보:
- 세션 ID
- 생성 시간
- 마지막 업데이트 시간
- 요약 정보 (있는 경우)

### 세션 상세 보기

세션 카드 클릭 시 대화 히스토리 표시:
- 사용자 메시지 (파란색)
- 어시스턴트 메시지 (회색)
- 도구 호출 결과 (접힌 상태로 표시)

### 실시간 업데이트

- WebSocket을 통한 실시간 메시지 감지
- 새 메시지 도착 시 자동 스크롤
- 상단 배지로 새 메시지 개수 표시

### 세션 검색

상단 검색창에서 키워드로 세션 검색:
- 세션 내용 기반 검색
- 프로젝트 필터링 가능

### 세션 재개

세션 상세 화면에서 재개 버튼 클릭:
- **Continue**: 기존 대화 이어서 진행
- **Resume**: 새로운 컨텍스트로 재시작

재개 시 Claude Code CLI가 새 터미널에서 실행됩니다.

### 백그라운드 에이전트

백그라운드 에이전트 탭에서 에이전트 실행 로그 확인:
- 에이전트 실행 상태
- 에이전트 출력 내용

## 2. 사용량 대시보드

토큰 사용량과 비용을 시각화합니다.

### 요약 통계 카드

상단에 표시되는 주요 지표:
- 총 비용 (USD)
- 총 세션 수
- 입력 토큰 수
- 출력 토큰 수

### 기간 필터

우측 상단 드롭다운으로 기간 선택:
- Last 7 Days: 최근 7일
- Last 30 Days: 최근 30일
- All Time: 전체 기간

### 모델별 분석

사용된 Claude 모델별 통계:
- 모델명
- 사용 횟수
- 토큰 사용량
- 비용

### 프로젝트별 분석

프로젝트별 사용량 분포:
- 프로젝트명
- 세션 수
- 총 비용
- 비율 (%)

### 일별 타임라인

날짜별 사용량 추이 차트:
- X축: 날짜
- Y축: 비용 또는 토큰 수
- 호버 시 상세 정보 표시

## 3. 프롬프트 분석

Claude를 활용한 지능형 프롬프트 패턴 분석입니다.

### 분석 대상 선택

1. 프로젝트 선택 (드롭다운)
2. 세션 선택 (체크박스)
3. 여러 세션 동시 선택 가능

### 분석 실행

"Analyze" 버튼 클릭:
1. 선택된 세션의 프롬프트 수집
2. 청크 단위로 분할 (대용량 처리)
3. Claude CLI를 통해 패턴 분석
4. 실시간 스트리밍으로 결과 표시

### 분석 결과

마크다운 형식으로 표시되는 내용:
- 주요 프롬프트 패턴
- 사용 빈도가 높은 키워드
- 작업 유형 분류
- 개선 제안

### 분석 기록 관리

- 분석 결과는 자동 저장
- 좌측 패널에서 이전 분석 조회
- 분석 삭제 및 재분석 가능

### 주의사항

- Claude Code CLI가 설치되어 있어야 합니다
- 분석에는 Claude API 토큰이 소모됩니다
- 대용량 세션은 분석 시간이 길어질 수 있습니다

## 4. 업무 분석

기간별 작업 내역을 종합 분석합니다.

### 날짜 범위 선택

1. 시작일 선택
2. 종료일 선택
3. "Load Sessions" 클릭

### 세션 조회 결과

프로젝트별로 그룹화된 세션 목록:
- 프로젝트명
- 세션 수
- 세션 목록 (펼치기/접기)

### 분석 모델 선택

분석에 사용할 Claude 모델 선택:
- Sonnet (기본)
- Opus
- Haiku

### 분석 실행

"Analyze Work" 버튼 클릭:
- 선택된 기간의 모든 세션 분석
- 작업 유형별 분류
- 시간 투자 패턴 파악

## 부가 기능

### 다국어 지원

우측 상단 언어 선택:
- 한국어
- English
- 日本語
- 中文

### 테마

시스템 설정에 따른 자동 테마 적용:
- 라이트 모드
- 다크 모드

### 캐시 관리

성능 최적화를 위한 캐싱:
- 프로젝트 목록: 30초
- 세션 목록: 30초
- 메타데이터: 60초
- 분석 결과: 120초

캐시는 API를 통해 수동 초기화 가능합니다.

## 키보드 단축키

| 단축키 | 동작 |
|--------|------|
| `Esc` | 모달 닫기 |
| `Ctrl/Cmd + K` | 검색 포커스 |

## 성능 최적화 팁

1. **세션 목록이 느린 경우**
   - 기간 필터를 좁게 설정
   - 프로젝트를 선택하여 범위 제한

2. **실시간 업데이트가 끊기는 경우**
   - 페이지 새로고침
   - WebSocket 연결 상태 확인

3. **분석이 오래 걸리는 경우**
   - 세션 선택 수를 줄임
   - 청크 처리 완료까지 대기
